# прописать в коде программы десять вопросов
  # на каждый вопрос необходимо дать ответ из одного слова
  # все вопросы и соответствующие им ответы должны быть сохранены в одну переменную
  # игровой процесс заключается в том, что программа по очереди выводит все вопросы, а игрок на каждый вопрос должен дать правильный ответ
  # предусмотреть возможность досрочного выход из игры

# после вопроса выводится число – количество букв в ответе, а также звёздочки в количестве равном количеству букв в ответе
  # при получении на вход неправильного ответа, программа выводит соответствущее сообщение и вместо одной из звёздочек подставляет букву, которая должна находится на этом месте в слове ответа – это подсказка
  # так продолжается до тех пор, пока игроком не будет дан правильный ответ или пока программа сама не откроет все буквы

# в игре ведётся подсчёт очков
  # за каждое угаданное слово к общему количеству очков прибавляется число, равное количеству букв в слове ответа
  # за каждую использованную подсказку из общего количества очков вычитается единица
  # общий счёт выводится: 
    # 1) после угаданного ответа, 
    # 2) после того, как закончатся все вопросы 
    # 3) когда игрок введёт кодовую фразу для выхода из игры
